# GEO Truth Engine

AI-powered fact verification and GEO visibility intelligence platform built for the Cog Culture assessment.

## Features
- PDF upload
- Claim extraction
- Verified / Inaccurate / False classification
- Hallucination detection
- Downloadable reports
- GEO visibility dashboard

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deploy on Streamlit Cloud.
